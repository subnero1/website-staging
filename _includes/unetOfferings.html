Network Development Platform & Simulator

For designing, prototyping and validating underwater network algorithms.

Unet Simulator is a network simulator that helps you simulate Unets with many nodes on a single computer. It can run in real-time simulation mode for interactive testing of agents and protocols, providing the same user experience as operating a real Unet. It also supports discrete-event simulation for large-scale or Monte Carlo performance evaluations.

Target Audience

For Users who want to design, test and validate subsea networking or MAC protocols.

Features
  • Multi-node, mobile network simulation
  • Rapid MAC, routing and cross-layer protocol prototyping
  • Seamless workflow: simulate → HIL → deploy

Hardware-in-the-Loop Underwater Communications Simulator

Avoid costly trial-and-error deployments by understanding performance under realistic conditions.

Virtual Acoustic Ocean is a high-fidelity, open-source underwater simulation platform supporting hardware-in-the-loop simulation for UnetStack-enabled devices. It interfaces with industry-standard channel models such as Bellhop, Kraken, Pekris Ray and upcoming benchmark Watermark channel models. This setup enables users to test algorithms under realistic underwater conditions using real hardware devices, including Subnero modems and UnetCube.

Target Audience

Users who want to test the performance of their algorithms in realistic underwater conditions, or simulate communication performance before field trials.

Features
  • Realistic acoustic propagation simulation
  • Supports simulation-only, UnetCube-HIL and modem-HIL workflows
  • Full open-source interface protocol enabling users to build on

OEM Modem Development Platform (UnetCube)

For OEMs building differentiated underwater modems or hybrid communication nodes.

UnetCube is an embedded platform that runs UnetStack natively, providing a software-defined modem engine. Users attach their own analog front-end and DAQ hardware to create differentiated modem products without rebuilding the entire communication stack.

Target Audience

Users who want to develop their own underwater modem.

Features
  • Runs UnetStack on embedded SBCs
  • Supports custom analog/DAQ integration
  • Unified workflow: simulate → HIL → deploy

Multi-Link Network Controller

For deploying scalable heterogeneous underwater networks.

A multi-link network controller that manages, prioritizes and routes data across different physical interfaces. Provides transparent end-to-end connectivity from seabed devices to topside systems and onward to cloud infrastructure.

Target Audience

Users deploying underwater networks for seamless connectivity across acoustic, optical and terrestrial links.

Features
  • Intelligent link selection and prioritization
  • Multi-modal routing (Acoustic/Optical/RF)
  • Seamless tracking and localization
